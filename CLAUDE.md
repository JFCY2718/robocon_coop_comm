# Claude Code Guide

你在这个项目中应优先完成可测试的小任务。

项目目标：实现 ROBOCON 2026「武林探秘」R1/R2 两机事件级通信，采用 AprilTag 定位 + LED 二进制光码。

请遵守：

- R1/R2 不使用无线射频通信。
- 武馆组装阶段不使用接触式通信。
- 遥控器只请求 R1 状态机切换，不直接控制 R2，不直接点 LED。
- 所有危险事件，例如 `INSERT_ALLOWED`，必须由 R1 本地传感器条件守卫。
- R2 收到消息后必须结合本地状态机和传感器条件判断。

开发顺序：

1. 保持 `pytest -q` 通过。
2. 修改协议时同步更新 `docs/PROTOCOL.md` 和测试。
3. 新增硬件模块时不要破坏纯软件 demo。
4. 每个状态机新增状态都要有单元测试。
